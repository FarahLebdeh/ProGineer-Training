
1. Write a shell script to calculate the sum of 1~100.

2. Using a shell script create directories from A-Z in root dir. Then copy all the directories in the /root/ directory to the /tmp/ directory.

3. Write a shell script which lists just the directories in the current directory (tmp)

For example in dir A there should be file A_file.txt and written the file should be “Hello A, welcome to shell!”
